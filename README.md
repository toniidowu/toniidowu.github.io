# Personal Portfolio Website

This repository contains the source code and design files for my professional portfolio. 

The project is built using R Markdown to integrate data analysis, interactive visualizations, and custom styling into a clean, professional web presence.

### Contents
- **index.Rmd**: Home page with professional introduction and photo
- **portfolio.Rmd**: Portfolio page featuring two public health data projects
- **resume.Rmd**: Resume page with embedded PDF
- **_site.yml**: Site navigation and configuration
- **styles.css**: Custom CSS for layout control and branding (project cards, flexbox layout, navbar styling)
- **resume.pdf**: Resume document
- **picture.png**: Profile photo
- **site_libs**: Required libraries and dependencies for the site
