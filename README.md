# Personal Portfolio Website

This repository contains the source code and design files for my professional portfolio. 

The project is built using R Markdown to integrate data analysis, interactive visualizations, and custom styling into a clean, professional web presence.

### Contents
* **index.Rmd / portfolio.Rmd**: Source files for the website content.
* **styles.css**: Custom CSS for layout control and branding (Project Cards, Flexbox layout).
* **site_libs**: Required libraries and dependencies for the site.
